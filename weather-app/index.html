This application is running on Docker Container, using Node and Express.Js.
It gets a Brazilian City's name from the user, fetch a Json from Accuweather API with all info about the current weather.
Then, it returns the Temperature, Condition, if it's day or night, and the Icon of the current condition.
For a better development maintance of the code, docker volume is being used with Nodemon for a quick reset of the server running on a AWS Virtual Machine.
